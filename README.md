Інтернет магазин + адмін панель CRM з аналітикою та автоматизацією складських залишків.

Frontend:
- React 
- Vite
- TypeScript
- Axios
- Socket.io-client — вебсокетити для роботи чатів підтримки.
- Recharts — Побудова графіків та діаграм в адмінці.
- hello-pangea/dnd — Бібліотека для Drag-and-Drop.
- React-i18next — підтримка декількох мов.
- Date-fns — робота з датами та форматуванням.
- Lucide-react

Backend 
- NestJS — потужний фреймворк.
- TypeScript
- Prisma ORM — ORM для взаємодії з базою даних.
- PostgreSQL — реляційна база даних.
- Socket.io (@nestjs/websockets) — cерверна частина для обміну повідомленнями в реальному часі.
- JWT (JSON Web Tokens) & Bcrypt — автентифікація, авторизація та хешування паролів.
- Multer — завантаження файлів.
- Class-validator & Class-transformer — валідація вхідних даних на сервері.
